## 11720 - 숫자의 합

### 문제 설명
N개의 숫자가 공백 없이 입력됨. 각 자릿수를 모두 더해서 출력하는 문제.

### 접근 방식
- 문자열로 받은 입력을 char 배열처럼 접근
- 각 문자를 -'0' 연산으로 숫자로 변환하여 합산

### 배운 점
- `char - '0'`을 이용해 문자 숫자를 int로 변환하는 방식 익힘
